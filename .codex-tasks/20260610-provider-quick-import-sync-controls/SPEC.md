# Provider Quick Import Sync Controls

## Goal

Move quick import sync settings out of the provider edit dialog into a provider-list action. Split upstream anomaly handling into per-condition policies and write administrator notifications with concrete sync change reasons.

## Scope

- Backend sync config type, storage mapping, migration SQL, sync outcome behavior, and admin notifications.
- Frontend provider row action, standalone sync settings dialog, and quick import source step defaults.
- Backend admin i18n seeds only.
