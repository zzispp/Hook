# 卡密管理功能

## 目标

实现管理员卡密管理：卡密类型、批量生成、导出 txt/csv、状态管理、批量启用/停用、按类型充值到用户钱包并记录流水。

## 边界

- 遵循现有 Rust workspace、Next.js 管理台、后端 i18n seed、菜单和 API 权限注册模式。
- 卡密类型由管理员维护，包含名称、充值余额、赠款余额、状态、备注。
- 卡密由管理员批量生成，选择类型，可设置长度、过期时间、状态、备注。
- 卡密兑换到账用户钱包，生成钱包流水。
- 管理界面参考钱包管理，两 tab：卡密列表、卡密类型，默认卡密列表。

## 验收

- 后端编译检查通过或暴露真实编译错误。
- 前端 lint/build 通过或暴露真实错误。
- i18n seed 包含新增后台文案和导航。
